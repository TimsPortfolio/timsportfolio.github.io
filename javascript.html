Gradient Maker

Copy the Code

This functions can draw rectangles and ellipses with gradents to color them instead of a solid color. Requires Processing JS.
Use the function with this code: gradient(type,x,y,width,height,startR,startG,startB,endR,endG,endB)

  1. type
      2. Select what shape you want your gradient to be (must be placed in quotes):
    • Horizontal: A rectangle with a gradient going from left to right.
    • Vertical: A rectangle with a gradient going from top to bottom.
    • Central: A rectangle with a gradient going from outside to inside.
    • Ellipse: An ellipse with a gradient going from outside to inside.
  2. x,y
      3. The x and y coordinate in the top left corner.
  3. width,height
      4. The width and height of the shape.
  4. startR,startG,startB
      5. The start color of the gradient in rgb
  5. endR,endG,endB
      6. The end color of the gradient in rgb

Grid Maker

Copy the Code

This function can draw a rectangular grid. Requires Processing JS.
Use the function with this code: grid(x,y,width,height,xSpace,ySpace)

  1. x,y
      2. The x and y coordinate of the top left of the grid.
  2. width,height
      3. The width and height of the grid.
  3. xSpace,ySpace
      4. The spacing between the lines in the grid starting at the top left.

To change the color and thickness of the grid, use the stroke, strokeWeight, and fill functions.
