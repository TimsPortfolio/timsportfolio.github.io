Gradient Maker

Get the Code

This functions can draw rectangles and ellipses with gradents to color them instead of a solid color. This is also the function that is being used to draw the sun setting animation above. Requires Processing JS.
Use the function with this code: gradient(type,x,y,width,height,startR,startG,startB,endR,endG,endB)

Type
Select what shape you want your gradient to be (must be placed in quotes):
Horizontal: A rectangle with a gradient going from left to right
Vertical: A rectangle with a gradient going from top to bottom.
Central: A rectangle with a gradient going from outside to inside.
Ellipse: An ellipse with a gradient going from outside to inside.
X,Y
The x and y coordinate in the top left corner.
Width,Height
The width and height of the shape.
startR,startG,startB
The start color of the gradient in rgb
endR,endG,endB
The end color of the gradient in rgbGrid Maker

Grid Maker

This function can draw a rectangular grid. Requires Processing JS.
Use the function with this code: grid(x,y,width,height,xSpace,ySpace)

x,y
The x and y coordinate of the top left of the grid.
width,height
The width and height of the grid.
xSpace,ySpace
The spacing between the lines in the grid starting at the top left.

To change the color and thickness of the grid, use the stroke, strokeWeight, and fill functions.
