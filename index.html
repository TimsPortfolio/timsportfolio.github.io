About this site

This website was built originally with just HTML, CSS, and Javascript. However it became a little cumbersome to do some things, esecially copy and paste the header and footer in every page, so I decided to use Nicepage. Nicepage is a drag and drop editor that allows a lot more flexibility then a traditionally drag and drop editor, which allows you to completely customize everything on your website.

You can place different elements on top of each other.

After I design the website it gives me all the code, which allows me to edit the code even more.
Right now it may be messy and boring because I just transfered everything over to nicepage. Soon I will go back and finish refining the pages.
